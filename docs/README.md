## 🎯 Projektziel

Eine vollständig **statische** Single-Page-Application (SPA) ohne Backend, die:
- Web-Links als anklickbare Icons darstellt
- Links in Ordnern organisiert (verschachtelbar)
- Einen immersiven „Desktop"-Look mit Glassmorphism & Animationen bietet
- Daten im `localStorage` des Browsers persistiert
- Als reine Build-Artefakte auf jedem Static-Host (GitHub Pages, Netlify, etc.) läuft
